axl_bool vortex_hash_delete ( VortexHash *  hash_table,
axlPointer  key 
)

Allows to remove the provided key and its associated data on the provided hash without calling to the optionally associated destroy functions.

Parameters:
hash_table The hash where the unlink operation will take place.
key The key for the data to be removed.
Returns:
axl_true if the item was removed (current implementation always return axl_true).
