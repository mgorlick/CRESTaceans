#define VORTEX_CHECK_REF2 ( ref,
return_value,
ref2,
free2_func   ) 
Value:
do { \
        if (ref == NULL) {                                          \
               free2_func (ref);                                    \
               return return_value;                                 \
        }                                                           \
} while (0)

Allows to check the reference provided, returning the return value provided, also releasing a second reference with a custom free function.

Parameters:
ref The reference to be checke for NULL.
return_value The return value to return in case of NULL reference.
ref2 Second reference to be released
free2_func Function to be used to release the second reference.
