void vortex_tls_set_default_post_check ( VortexCtx *  ctx,
VortexTlsPostCheck  post_check,
axlPointer  user_data 
)

Allows to configure a function that will be executed at the end of the TLS process, before returning the connection to the application level.

See VortexTlsPostCheck for more information.

If you want to configure a per-connection handler you can use: vortex_tls_set_default_post_check

Parameters:
ctx The context where the operation will be performed.
post_check The handler to be called once required to perform the post-check.
user_data User defined data, that is passed to the handler provided (post_check).
