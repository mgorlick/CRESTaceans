15 Persistent unordered sets

syntax

set/eq/null

Produces a new persistent set for which (set/empty? set) is equal to #t, therefore an empty persistent set where elements are compared with eq?.

syntax

set/eqv/null

Produces a new persistent set for which (set/empty? set) is equal to #t, therefore an empty persistent set where elements are compared with eqv?.

syntax

set/equal/null

Produces a new persistent set for which (set/empty? set) is equal to #t, therefore an empty persistent set where elements are compared with equal?.

procedure

(set/persist? e) → boolean?

  e : any/c
Returns #t if v is a persistent set, #f otherwise.

procedure

(set/eq? s) → boolean?

  s : set/persist?
Returns #t if the equality test for elements is eq?, #f otherwise.

procedure

(set/eqv? s) → boolean?

  s : set/persist?
Returns #t if the equality test for elements is eqv?, #f otherwise.

procedure

(set/equal? s) → boolean?

  s : set/persist?
Returns #t if the equality test for elements is equal?, #f otherwise.

procedure

(list/set s lst) → set/persist?

  s : set/persist?
  lst : list?
Nondestructively produces a new persistent set |hash'| from the union of set s and a list of elements (e_0 ... e_{n-1}).

procedure

(set/new s e ...) → set/persist?

  s : set/persist?
  e : any/c
Equivalent to (list/set s lst), but takes multiple arguments instead of a list of elements.

procedure

(vector/set s vec) → set/persist?

  s : set/persist?
  vec : vector?
Non-destructively creates a set |s'| that is the union of the contents of set s and the contents of vector vec = #(e_0 e_1 ... e_ {n-1}).

procedure

(set/cons s e) → set/persist?

  s : set/persist?
  e : any/c
Produces a new persistent set |s'| by adding the given value to the set.

procedure

(set/remove s e) → set/persist?

  s : set/persist?
  e : any/c
Produces a new persistent set |s'| whose contents are identlical to s but without the element indexed by e.

procedure

(set/union s1 s2) → set/persist?

  s1 : set/persist?
  s2 : set/persist?
Produces a new persistent set |s1'| that is the union of set s1 and s2. The resulting set is defined with the equality? and hash functions of set s1.

procedure

(set/intersection s1 s2) → set/persist?

  s1 : set/persist?
  s2 : set/persist?
Produces a new persistent set |s1'| that is the intersection of set s1 and s2. The resulting set is defined with the equality? and hash functions of set s1.

procedure

(set/difference s1 s2) → set/persist?

  s1 : set/persist?
  s2 : set/persist?
Returns the subset |s1'| of s1 such that no member of |s1'| is also a member of s2.

procedure

(set/list s) → list?

  s : set/persist?
Returns a list of the elements of s in an unspecified order.

procedure

(set/vector s) → vector?

  s : set/persist?
Returns a vector #(e_0 ... e_{n-1}) of elements within s.

procedure

(set/car s) → any/c

  s : set/persist?
Returns the "first" element of set s.

procedure

(set/cdr s) → list?

  s : set/persist?
Produces a new persistent set |s'| in which the element returned by (set/car hash) is absent.

procedure

(set/length s) → natural/exact?

  s : set/persist?
Returns the length of s (i.e., the number of elements in the persistent set).

procedure

(set/empty? s) → boolean?

  s : set/persist?
Returns #t if (set/length s) is equal to 0, otherwise returns #f.

procedure

(set/contains? s e) → boolean?

  s : set/persist?
  e : any/c
Returns #t if s contains element e and #f otherwise.

procedure

(set/fold s proc seed) → any/c

  s : set/persist?
  proc : procedure?
  seed : any/c
Fold function proc with seed over all pairs in persistent set s. Function proc takes two arguments, an element of set s and the current seed, and returns the new seed value. The value of the fold is the final seed value.

procedure

(set/map s proc) → set/persist?

  s : set/persist?
  proc : procedure?
Applies proc to each element in s. The proc function takes a single argument. The result is a persistent set containing the results.

procedure

(set/subset? s1 s2) → booleant?

  s1 : set/persist?
  s2 : set/persist?
Returns #t if set s2 is a subset of set s1 and #f otherwise.}

procedure

(set/filter s proc) → set/persist?

  s : set/persist?
  proc : procedure?
Returns a persistent set containing only those elements in s for which (proc e) is #t.

procedure

(set/partition s proc) → pair?

  s : set/persist?
  proc : procedure?
Partitions set s into two disjoint subsets s1 and s2 of s where (proc e) is #t for all members of s1 and returns #f for all members of s2. Returns the partition as a pair (s1 . s2).
