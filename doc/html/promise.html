24 Promise

procedure

(promise/new lifespan) → pair?

  lifespan : (and/c number? positive?)
Returns the pair (p . c) where p is the promise and c is the CURL to which the resolution of p should be sent. lifespan is the maximun length of the promise expressed in seconds.

procedure

(promise? v) → boolean?

  v : any/c
Returns #t if v is a promise, #f otherwise.

procedure

(promise/kept? p) → boolean?

  p : promise?
Returns #t if promise p has been resolved, #f otherwise.

procedure

(promise/ruined? p) → boolean?

  p : promise?
Returns #t if promise p has expired before its resolution, #f otherwise.

procedure

(promise/wait p patience failure) → any/c

  p : promise?
  patience : (and/c number? positive?)
  failure : any/c
Waits on promise p. Patience p is the time (a postive number) giving the span of time (expressed in real seconds) that the calling thread (actor) is willing to wait for the resolution of promise p. failure is the value returned if promise p is not kept by the caller’s deadline (in patience seconds), or if promise p has been ruined. Returns the value of promise p if the promise was kept and returns failure if the promise has been ruined or if the caller’s patience is exhausted.

procedure

(promise/call p f patience failure) → any/c

  p : promise?
  f : procedure?
  patience : (and/c number? positive?)
  failure : any/c
Returns the value of applying function f to the outcome of promise p. Patience p is the time for resolution of promise p expressed in real seconds. failure is the value to be substituted if promise p is not kept in patience seconds.
