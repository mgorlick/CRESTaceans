19 Islands

procedure

(island/address/new public_key dns ip_port) → island?

  public_key : bytes?
  dns : bytes?
  ip_port : (and/c integer? positive?)
Creates a new self-certifying island given its public key, DNS name, and IP port. A usage example is (island/address/new #"abcd" #"127.0.0.1" 5000).

procedure

(island/address? v) → boolean?

  v : any/c
Returns #t if v is an island, #f otherwise.

procedure

(island/address/ok? i) → boolean?

  i : island/address?
Returns #t if the fields of island/address i are type correct, #f otherwise.

procedure

(island/address/equal? i1 i2) → boolean?

  i1 : island/address?
  i2 : island/address?
Returns #t island/address i1 is equal to island/address i2, #f otherwise.

procedure

procedure

(this/island) → any/c
