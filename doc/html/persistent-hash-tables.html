13 Persistent functional hash tables

syntax

hash/eq/null

Produces a new persistent hash table for which (hash/empty? hash) is equal to #t, therefore an empty persistent hash table where keys are compared with eq?.

syntax

hash/eqv/null

Produces a new persistent hash table for which (hash/empty? hash) is equal to #t, therefore an empty persistent hash table where keys are compared with eqv?.

syntax

hash/equal/null

Produces a new persistent hash table for which (hash/empty? hash) is equal to #t, therefore an empty persistent hash table where keys are compared with equal?.

procedure

(hash/persist? v) → boolean?

  v : any/c
Returns #t if v is a persistent hash table, #f otherwise.

procedure

(hash/eq? hash) → boolean?

  hash : hash/persist?
Returns #t if the equality test for keys is eq?, #f otherwise.

procedure

(hash/eqv? hash) → boolean?

  hash : hash/persist?
Returns #t if the equality test for keys is eqv?, #f otherwise.

procedure

(hash/equal? hash) → boolean?

  hash : hash/persist?
Returns #t if the equality test for keys is equal?, #f otherwise.

procedure

(list/hash hash lst) → hash/persist?

  hash : hash/persist?
  lst : list?
Nondestructively produces a new persistent hash table |hash'| from the union of the contents of hash and a list of key/value pairs given as input in the form of (k_0 v_0 ... ... k_i v_i). If hash contains a pair k_i/x for some value x then it will be replaced by k_i/v_i in |hash'|.

procedure

(hash/new hash key v ... ...) → hash/persist?

  hash : hash/persist?
  key : any/c
  v : any/c
Equivalent to (list/hash hash lst), but takes multiple arguments instead of a list of key/value pairs.

procedure

(pairs/hash hash lst) → hash/persist?

  hash : hash/persist?
  lst : list?
Equivalent to (list/hash hash lst), but takes as input a list of lists in the form of ((k_0 . v_0) ... (k_i . v_i)).

procedure

(vector/hash hash vec) → hash/persist?

  hash : hash/persist?
  vec : vector?
Non-destructively creates a fresh hash table |hash'| that is the union of the contents of the hash table hash and the contents of vector vec = #(k_0 v_0 k_1 v_1 ... k_ {n-1} v_ {n-1}). Equivalent of (pairs/hash h ((k_0 . v_0) ... (k_i . v_i))).

procedure

(vectors/hash hash keys values) → hash/persist?

  hash : hash/persist?
  keys : vector?
  values : vector?
Non-destructively creates a hash table |hash'| that is the union of the contents of the hash table hash and the contents of vectors keys = #(k_0 k_1 ... k_ {n-1}) corresponding to the keys and keys = #(v_0 v_1 ... v_ {n-1}) corresponding to the respective keys’ values.

procedure

(hash/cons hash key v) → hash/persist?

  hash : hash/persist?
  key : any/c
  v : any/c
Produces a new persistent hash table |hash'| by merging the contents of hash and the given key/value pair. If hash contains key then its value is replaced by v in the |hash'|.

procedure

(hash/remove hash key) → hash/persist?

  hash : hash/persist?
  key : any/c
Produces a new persistent hash table |hash'| whose contents are identlical to hash but without the key/value pair indexed by key.

procedure

(hash/vector/remove hash keys) → hash/persist?

  hash : hash/persist?
  keys : vector?
Produces a new persistent hash table |hash'| whose contents are identlical to hash but without the key/value pairs indexed by the contents of vector keys.

procedure

(hash/merge hash1 hash2) → hash/persist?

  hash1 : hash/persist?
  hash2 : hash/persist?
Produces a new persistent hash table |hash1'| that is the merge of hash2 INTO hash1. Any key/value pair in hash2 whose key duplicates a key appearing in hash1 will replace the corresponding key/value pair in hash1. The length of the resulting hash table will be at most (+ (hash/length hash1) (hash/length hash2)).

procedure

(hash/list hash) → list?

  hash : hash/persist?
Returns a list of the values of hash in an unspecified order.

procedure

(hash/pairs hash) → list?

  hash : hash/persist?
Returns an association list ((k_0 . v_0) ...) of key/value pairs within hash.

procedure

(hash/vector hash) → vector?

  hash : hash/persist?
Returns a vector #(k_0 v_0 ... k_{n-1} v_{n-1}) of key/value pairs within hash.

procedure

(hash/keys hash) → list?

  hash : hash/persist?
Returns the set of keys in hash as a list.

procedure

(hash/values hash) → list?

  hash : hash/persist?
Returns the set of values in hash as a list.

procedure

(hash/ref hash key failure) → any/c

  hash : hash?
  key : any/c
  failure : any/c
Returns the value for key in hash. If no value is found for key, then failure is returned as the result. failure can as well be a procedure that is called in case key is not in hash.

procedure

(hash/car hash) → any/c

  hash : hash/persist?
Returns the "first" key/value pair of hash table hash.

procedure

(hash/cdr hash) → list?

  hash : hash/persist?
Produces a new persistent hash table |hash'| in which the key/value pair returned by (hash/car hash) is absent. Equivalent to (hash/remove hash (car (hash/car hash))) but considerably more efficient.

procedure

(hash/length hash) → natural/exact?

  hash : hash/persist?
Returns the length of hash (i.e., the number of key/value pairs in the persistent hash table).

procedure

(hash/empty? hash) → boolean?

  hash : hash/persist?
Returns #t if (hash/length hash) is equal to 0, otherwise returns #f.

procedure

(hash/contains? hash key) → boolean?

  hash : hash/persist?
  key : any/c
Returns #t if hash contains key and #f otherwise.

procedure

(hash/fold hash proc seed) → any/c

  hash : hash/persist?
  proc : procedure?
  seed : any/c
Fold function proc with seed over all pairs in persistent hash table hash.

procedure

(hash/map hash proc) → hash/persist?

  hash : hash/persist?
  proc : procedure?
Applies proc to each key/value pair in hash. The proc function takes a single argument. The result is a persistent hash map containing the results.

procedure

(hash/for-each hash proc) → void?

  hash : hash/persist?
  proc : procedure?
Applies proc to each key/value pair in hash and discards all results. The proc function takes a single argument.

procedure

(hash/filter hash proc) → hash/persist?

  hash : hash/persist?
  proc : procedure?
Returns a persistent hash table containing only those key/value pairs in hash for which (proc pair) is #t.

procedure

(hash/partition hash proc) → hash/persist?

  hash : hash/persist?
  proc : procedure?
Returns a pair of hash tables (hash-true . hash-false) in which hash table hash-true contains only those pairs of hash for which predicate proc returns #t and hash table hash-false contains only those pairs of hashfor which predicate proc returns #f.
