25 ISLAND - Peering infrastructure

procedure

(listen-on/make-root i)

 → 
actor? locative? locative? curl? thread?
  i : island/address?
Sets the island address to i, creates a root actor, creates and registers a public locative in island i’s locative serialization table, retrieves a public locative and public CURL for use in speaking to the root actor, and starts up the networking layer listening on that island address. Returns a set of values which includes the root actor, its locative, a public locative, and a public curl.

procedure

(eval-actor a f env) → void?

  a : actor?
  f : procedure?
  env : environ?
Given an actor a, a motile zero-argument procedure f, and a binding environment env, instantiate a as the procedure f evaluated in the context of env.

procedure

(argsassoc key    
  [#:call call    
  #:no-val no-val    
  #:default default]) → any/c
  key : any/c
  call : procedure? = values
  no-val : any/c = #f
  default : any/c = #t
Separates the provided command line arguments of the form key1=val1 key2=val2 key3=val3. If the provided key name is present and has a value, returns call applied to that value. If the provided key name is present but has no value, returns ‘default.If the provided key name is not present, returns no-val.
