1 Type testing

(boolean? v) → boolean?
  v : any/c
Returns #t if v is #t or #f, #f otherwise.

(list? v) → boolean?
  v : any/c
Returns #t if v is a list: either the empty list, or a pair whose second element is a list. Otherwise, it returns #f.

(null? v) → boolean?
  v : any/c
Returns #t if v is the empty list, #f otherwise.

(pair? v) → boolean?
  v : any/c
Returns #t if v is a pair, #f otherwise.

(symbol? v) → boolean?
  v : any/c
Returns #t if v is a symbol, #f otherwise.

(char? v) → boolean?
  v : any/c
Return #t if v is a character, #f otherwise.

(string? v) → boolean?
  v : any/c
Returns #t if v is a string, #f otherwise.

(byte? v) → boolean?
  v : any/c
Returns #t if v is a byte (i.e., an exact integer between 0 and 255 inclusive), #f otherwise.

(bytes? v) → boolean?
  v : any/c
Returns #t if v is a byte string, #f otherwise.

(number? v) → boolean?
  v : any/c
Returns #t if v is a number, #f otherwise.

(real? v) → boolean?
  v : any/c
Returns #t if v is a real number, #f otherwise.

(integer? v) → boolean?
  v : any/c
Returns #t if v is an integer number, #f otherwise.

(exact? z) → number?
  z : any/c
Returns #t if z is an exact number, #f otherwise.

(inexact? z) → number?
  z : any/c
Returns #t if z is an inexact number, #f otherwise.

(zero? z) → boolean?
  z : number?
Returns (= 0 z).

(positive? x) → boolean?
  x : real?
Returns (> x 0).

(negative? x) → boolean?
  x : real?
Returns (< x 0).

(even? n) → boolean?
  n : integer?
Returns (zero? (modulo n 2)).

(odd? n) → boolean?
  n : integer?
Returns (not (even? n)).

(procedure? v) → boolean?
  v : any/c
Returns #t if v is a procedure, #f otherwise.
