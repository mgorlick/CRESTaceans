14 Persistent functional records

procedure

(record? v) → boolean?

  v : any/c
Returns #t if v is a record, #f otherwise. A record is a five-element vector where the 1st element is ’record and the 3rd element is a persistent hash table, i.e. (vector ’record <kind> <hash> <signature>).

procedure

(record/contains? rec key) → boolean?

  rec : record?
  key : any/c
Returns #t if record rec contains a given key, #f otherwise.

procedure

(record/signed? rec) → boolean?

  rec : record?
Returns #t if record rec is digitally signed (for the purpose of verifying that the record has not been changed), #f otherwise. Signature is in the form of a byte string.

procedure

(record/kind rec) → symbol?

  rec : record?
Returns the kind of record rec as a symbol.

procedure

(record/keys rec) → list?

  rec : record?
Return the field names of record rec as a list of symbols.

procedure

(record/unsign rec) → record?

  rec : record?
Produces an unsigned copy of record rec.
