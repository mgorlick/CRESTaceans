6 Arithmetic operations

procedure

(+ z ...) → number?

  z : number?
Returns the sum of the zs, adding pairwise from left to right. If no arguments are provided, the result is 0.

procedure

(- z) → number?

  z : number?
(- z w ...+) → number?
  z : number?
  w : number?
When no ws are supplied, returns (- 0 z). Otherwise, returns the subtraction of the ws from z working pairwise from left to right.

procedure

(* z ...) → number?

  z : number?
Returns the product of the zs, multiplying pairwise from left to right. If no arguments are provided, the result is 1. Multiplying any number by exact 0 produces exact 0.

procedure

(/ z) → number?

  z : number?
(/ z w ...+) → number?
  z : number?
  w : number?
When no ws are supplied, returns (/ 1 z). Otherwise, returns the division of z by the ws working pairwise from left to right.

If z is exact 0 and no w is exact 0, then the result is exact 0. If any w is exact 0, an exception will be raised.

procedure

(quotient n m) → integer?

  n : integer?
  m : integer?
Returns (truncate (/ n m)).

procedure

(remainder n m) → integer?

  n : integer?
  m : integer?
Returns q with the same sign as n such that

  • (abs q) is between 0 (inclusive) and (abs m) (exclusive), and

  • (+ q (* m (quotient n m))) equals n.

If m is exact 0, an exception will be raised.

procedure

(modulo n m) → integer?

  n : integer?
  m : integer?
Returns q with the same sign as m where

  • (abs q) is between 0 (inclusive) and (abs m) (exclusive), and

  • the difference between q and (- n (* m (quotient n m))) is a multiple of m.

If m is exact 0, an exception will be raised.

procedure

(add1 z) → number?

  z : number?
Returns (+ z 1).

procedure

(sub1 z) → number?

  z : number?
Returns (- z 1).

procedure

(1- z) → number?

  z : number?
A handy alias for add1.

procedure

(1+ z) → number?

  z : number?
A handy alias for sub1.

procedure

(exp z) → number?

  z : number?
Returns Euler’s number raised to the power of z. The result is normally inexact, but it is exact 1 when z is an exact 0.

procedure

(log z) → number?

  z : number?
Returns the natural logarithm of z. The result is normally inexact, but it is exact 0 when z is an exact 1. When z is exact 0, an exception will be raised.

procedure

(sqrt z) → number?

  z : number?
Returns the principal square root of z. The result is exact if z is exact and z’s square root is rational. See also integer-sqrt.

procedure

(expt z w) → number?

  z : number?
  w : number?
Returns z raised to the power of w. If w is exact 0, the result is exact 1. If z is exact 0 and w is negative, an exception will be raised.

procedure

(max x ...+) → real?

  x : real?
Returns the largest of the xs, or +nan.0 if any x is +nan.0. If any x is inexact, the result is coerced to inexact.

procedure

(min x ...+) → real?

  x : real?
Returns the smallest of the xs, or +nan.0 if any x is +nan.0. If any x is inexact, the result is coerced to inexact.

procedure

(gcd n ...) → integer?

  n : integer?
Returns the greatest common divisor (a non-negative number) of the ns. If no arguments are provided, the result is 0. If all arguments are zero, the result is zero.

procedure

(lcm n ...) → integer?

  n : integer?
Returns the least common multiple (a non-negative number) of the ns. If no arguments are provided, the result is 1. If any argument is zero, the result is zero; furthermore, if any argument is exact 0, the result is exact 0.

procedure

(round x) → integer?

  x : real?
Returns the integer closest to x, resolving ties in favor of an even number.

procedure

(floor x) → integer?

  x : real?
Returns the largest integer that is no more than x.

procedure

(ceiling x) → integer?

  x : real?
Returns the smallest integer that is at least as large as x.

procedure

(truncate x) → integer?

  x : real?
Returns the integer farthest from 0 that is not farther from 0 than x.

procedure

(numerator q) → integer?

  q : rational?
Coerces q to an exact number, finds the numerator of the number expressed in its simplest fractional form, and returns this number coerced to the exactness of q.

procedure

(denominator q) → integer?

  q : rational?
Coerces q to an exact number, finds the numerator of the number expressed in its simplest fractional form, and returns this number coerced to the exactness of q.

procedure

(rationalize x tolerance) → real?

  x : real?
  tolerance : real?
Among the real numbers within (abs tolerance) of x, returns the one corresponding to an exact number whose denominator is the smallest. If multiple integers are within tolerance of x, the one closest to 0 is used.
