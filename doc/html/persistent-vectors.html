12 Persistent functional vectors

vector/null
Produces a new persistent vector for which (vector/length vec) is equal to 0, therefore and empty vector.

(vector/null? v) → boolean?
  v : any/c
Returns #t if v is equal to vector/null, #f otherwise.

(vector/persist? v) → boolean?
  v : any/c
Returns #t if v is a persistent vector (an immutable data structure), #f otherwise.

(vector/build n proc) → vector/persist?
  n : natural/exact?
  proc : (natural/exact? . -> . any/c)
Creates a persistent vector of n elements by applying proc to the integers from 0 to (sub1 n) in order. If vec is the resulting immutable vector, then (vector/ref vec i) is the value produced by (proc i).

(vector/length vec) → natural/exact?
  vec : vector/persist?
Returns the length of vec (i.e., the number of slots in the vector).

(vector/ref vec pos) → any/c
  vec : vector/persist?
  pos : natural/exact?
Returns the element in slot pos of vec. The first slot is position 0, and the last slot is one less than (vector/length vec).

(list/vector vec lst) → vector/persist?
  vec : vector/persist?
  lst : list?
Returns a new immutable or persistent vector that contains the values of vec and the values of lst. The values of lst are appended, in the newly created vector, to the tail of vec. vec remains unchanged. If vec vector/length is 0 the new persistent vector’s lenght is the same as lst lenght, otherwise the new persistent vector’s lenght is equal to the lenght of vec plus the length of the list.}

(vector/fold/left vec proc init) → any/c
  vec : vector/persist?
  proc : procedure?
  init : any/c
vector/fold/left applies a procedure to the elements of a persistent vector, combining the return values in an arbitrary way determined by proc. proc must be a function that takes at least 2 arguments. On the first iteration, proc takes as arguments init and the first element on the vector. The input vector is traversed from left to right, and the result of the whole vector/fold/left is the result of the last application of proc. If the vec is empty, the result is init.

(vector/fold/right vec proc init) → any/c
  vec : vector/persist?
  proc : procedure?
  init : any/c
Like vector/fold/left, but the vector vec is traversed from right to left.

(vector/map vec proc) → vector/persist?
  vec : vector/persist?
  proc : procedure?
Applies proc to the elements of vec from the first elements to the last. The proc function is an unary one, thus takes a single argument. The result is a fresh vector containing each result of proc in order.

(vector/filter vec pred) → vector/persist?
  vec : vector/persist?
  pred : procedure?
Returns a fresh persisntent vector with the elements of vec for which pred produces a true value. The pred procedure is applied to each element from first to last.

(vector/cons vec v) → vector/persist?
  vec : vector/persist?
  v : any/c
Appends v at the end of vector vec.

(vector/subvector vec from) → vector/persist?
  vec : vector/persist?
  from : natural/exact?
Returns a new persistent vector with the subset of vec’s elements starting from index from up to the last index of vec.

(vector/subvector vec from end) → vector/persist?
  vec : vector/persist?
  from : natural/exact?
  end : natural/exact?
Returns a new persistent vector with the subset of vec’s elements starting from index from up to index of end of vector vec.

(vector/cdr vec) → vector/persist?
  vec : vector/persist?
Removes the tail (last) element of persistent vector vec.

(vector/update vec i v) → vector/persist?
  vec : vector/persist?
  i : natural/exact?
  v : any/c
Sets the value at index i of vector vec to v.
